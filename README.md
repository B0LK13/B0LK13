## TBA thema-configuratie

Het standaardthema staat nu op **TBA**. De kleuren staan in `themes.js` en worden via `BLOG_THEME=tba` geactiveerd (dit is de default). Het logo staat in `public/images/tba-logo.svg`; je kunt een eigen logo instellen met de omgevingsvariabele `BLOG_LOGO`.
